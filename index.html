Basics of the web: Notes from Day 1 - Project 0 HTML stands for Hyper Text Markup Language,
and the web was created in the 1990s with around 30 billion pages!

It basically works like this: The end user, me, you, computer, etc. sends a request via http through the internet to a server, that server then grants the request and sends it back through the internet to me.



HTML is the heart of the web and is made up of various parts such as:
Text content:what you see
Mark-up:what it looks like
References to other documents:e.g. images and videos
Links to other pages.

We also learn the difference between inline and block elements.
Inline does not create a new block of content, such as the span tag, or the break tag, where as Block will create a new block of content, such as the paragraph tag, or the div tag. Block elements form an "invisible box" around the content inside of them.
